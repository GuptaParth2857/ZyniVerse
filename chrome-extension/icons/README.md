# Icons

Generate icons by opening `../generate-icons.html` in a browser and taking screenshots at 16x16, 48x48, and 128x128.

## Placeholder Icons

The current PNG files are 1x1 transparent placeholders. Replace them with proper icons before publishing.

## Requirements

- **icon16.png** — 16x16px, used in browser toolbar
- **icon48.png** — 48x48px, used in extension management page
- **icon128.png** — 128x128px, used in Chrome Web Store

## Design

- Background: `#0a0a0f` (dark)
- Letter "Z" in purple: `#c084fc`
- Rounded square shape with gradient
