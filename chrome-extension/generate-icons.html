How to generate PNG icons

  1. Open this page in Chrome.
  2. Right-click each icon and select "Save image as..." (not available — use screenshot method instead).
  3. Alternatively, take a screenshot and crop each icon to the exact pixel size.
  4. Save as icon16.png, icon48.png, and icon128.png in the icons/ folder.
